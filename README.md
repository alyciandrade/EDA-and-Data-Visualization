# Português
Projeto de Análise Exploratória de Dados (EDA) e Visualização de Dados utilizando o dataset **Top Spotify Songs 2023**. O objetivo é explorar características musicais, popularidade, artistas e padrões presentes nas músicas mais ouvidas do ano, aplicando técnicas de limpeza, transformação, análise estatística e visualizações para geração de insights. O projeto foi desenvolvido em Python, com uso de bibliotecas como Pandas, Matplotlib e Seaborn.
> Para mais detalhes, acesse a [Wiki do repositório](../../wiki).

# English
This project focuses on Exploratory Data Analysis (EDA) and Data Visualization using the **Top Spotify Songs 2023** dataset. The goal is to explore musical features, popularity, artists, and patterns among the most streamed songs of the year by applying data cleaning, transformation, statistical analysis, and visualization techniques to generate insights. The project was developed in Python, using libraries such as Pandas, Matplotlib, and Seaborn.
> For more details, please visit the [repository Wiki](../../wiki).
